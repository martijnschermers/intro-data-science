# Intro Data Science
Assignments for Intro Data Science

## Prequisites
- Git LFS, used for storing the large CSV files.

## Run Locally

- Install [Git LFS](https://git-lfs.com/)
- Run the following command to install the CSV files: 
```bash
    git lfs pull
    git lfs fetch
```

- Start the notebook with: 
```bash
    jupyter notebook
```
